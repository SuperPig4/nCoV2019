#### 介绍
获得疫情最新状况<br/>
第一时间获得疫情消息<br/>

#### 软件架构
python3<br>
sqlite3<br>

#### 安装教程
pip install -r requirements.txt<br>

#### 使用说明
Main.py update 更新数据<br>
Main.py send   发送邮件<br>
Main.py aoto   自动(更新数据->发送邮件),一般定时执行这个<br>
Server.py      启动http服务
